# PaddiSense Modules

Released modules for PaddiSense Farm Management.

This repository is pulled by the PaddiSense installer on grower servers.

## Installation

Modules are installed via the PaddiSense custom component. 
See [PKmac78/PaddiSense](https://github.com/PKmac78/PaddiSense) for installation instructions.

## Available Modules

See `PaddiSense/modules.json` for the current release manifest.
